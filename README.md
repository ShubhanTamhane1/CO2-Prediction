# 🚗 CO₂ Emission Prediction using Random Forest Regressor

This project builds a **Random Forest Regressor** to predict the amount of **CO₂ emitted by a car** based on its specifications. The model is trained using a dataset containing vehicle attributes like engine size, fuel consumption, and other relevant features.

## 📌 Features
- Uses **Random Forest Regressor** for accurate CO₂ emission predictions.
- Performs **data preprocessing and feature engineering** for improved model performance.
- Includes **exploratory data analysis (EDA)** for understanding feature impact.
- Evaluates model performance using **Mean Squared Error (MSE), R² Score, and MAE**.

## 📡 Technologies Used
- **Programming Language: R**
- Libraries and Tools:
  - Dplyr
  - Car
  - RandomForest
  - ggplot2 
  - R Studio
  - R Markdown

## 📂 Dataset
The dataset includes the following features:
- **Engine Size (L)**
- **Fuel Type**
- **Vehicle Class**
- **Vehicle Make**
- **Transmission Type**
- **CO₂ Emissions (g/km) [Target]**

 **Source**: *https://www.kaggle.com/datasets/brsahan/vehicle-co2-emissions-dataset/data*


## 📊 Results:
- **MAE:** 12.45
- **MSE:** 210.89
- **R² Score:** 0.92
